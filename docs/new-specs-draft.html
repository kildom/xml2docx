<p

Paragraph.

The paragraph contains formatted text and images.

Any whitespaces at the beginning and end of the paragraph are removed. If you want to keep the whitespaces, enclose the text in a <font> tag.

The paragraph expects <font> tag as a child. If it contains any other tags, they will be implicitly enclosed in a <font> tag.

⊞  align="alignment"
⊞  border="top, right, bottom, left"
⊞  tabs="tab-stop, tab-stop, ..."
>
⊟  Allowed children:
      <font/> (implicit)
      Allowed children of implicit tag:
      <img/> <b/> <i/> <u/> <s/> <sub/> <sup/> text
</p> 


* Tag description is always expanded.
* Attributes and children are collapsed by default.
* Hovering over attribute type will show a tooltip with the type description.
* Only most common enum values are shown for attributes, rest is available after clicking "..." button.
* There should be global "Expand all" and "Collapse all" buttons useful for searching.
* Attributes should be sorted by importance.
